# Customer Churn Analysis Dashboard

## Overview
This project focuses on analyzing customer churn data to understand why customers stop using a service. The dashboard helps identify churn patterns and supports better customer retention strategies.

## Objective
- Measure churn rate
- Identify customer segments with high churn risk
- Understand the impact of contract type, tenure, and payment behavior on churn

## Tools & Technologies
- Power BI
- Excel / CSV
- Data Cleaning
- Data Visualization

## Dataset
The dataset includes customer information such as:
- Gender
- Senior Citizen
- Tenure
- Contract Type
- Monthly Charges
- Payment Method
- Internet Service
- Churn Status

## Project Workflow
1. Imported customer churn dataset
2. Performed data cleaning and transformation
3. Created KPI measures
4. Designed interactive dashboard visuals
5. Generated insights for retention improvement

## Key KPIs
- Total Customers
- Churned Customers
- Churn Rate
- Average Monthly Charges
- Average Tenure

## Dashboard Highlights
- Churn by Contract Type
- Churn by Payment Method
- Churn by Tenure Group
- Churn by Internet Service
- Monthly Charges vs Churn
- Demographic-based churn analysis

## Key Insights
- Customers on month-to-month contracts have higher churn
- Short-tenure customers are more likely to leave
- Certain payment methods show higher churn behavior
- Higher monthly charges may contribute to churn in some segments

## Business Recommendations
- Create retention offers for short-tenure customers
- Encourage long-term contracts
- Improve service quality for high-risk segments
- Offer pricing or loyalty benefits to vulnerable customer groups
