# Customer Churn Analysis

## Project Overview
This project analyzes customer churn data to understand the key factors influencing customer retention and churn behavior.

## Tools Used
- SQL
- Power BI

## Key Metrics
- Total Customers
- Churn Rate
- Contract Type Analysis
- Payment Method Analysis

## Insights
- Customers with month-to-month contracts have higher churn rates.
- Higher monthly charges are associated with increased churn probability.
- Customers using electronic payment methods showed higher churn risk.
